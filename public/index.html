                             API לחיבור חיצוני
                        

                        

                            באפשרותך לשלוח הודעות דרך ה-API באמצעות בקשת HTTP POST מכל מערכת חיצונית.
                        


                        

                            
                            
curl -X POST http://localhost:3000/api/send-message \
  -H "Content-Type: application/json" \
  -d '{
    "sessionId": "default",
    "to": "972501234567",
    "message": "Hello from Shargo Bot!"
  }'

                        


                        

                            
פרמטרים נתמכים:

                            
    
                                
  • sessionId: מזהה החשבון השולח.
    • 
                                
  • to: מספר יעד (972...).
    • 
                                
  • message: תוכן ההודעה (טקסט חופשי).
    •

  -H "Content-Type: application/json" \

  }'
