                             שליחת הודעה ידנית

                             API לחיבור חיצוני
                        

                        

                            באפשרותך לשלוח הודעות דרך ה-API באמצעות בקשת HTTP POST מכל מערכת חיצונית.
                        


                        

                            
                            
curl -X POST http://localhost:3000/api/send-message \
  -H "Content-Type: application/json" \
  -d '{
    "sessionId": "default",
    "to": "972501234567",
    "message": "Hello from Shargo Bot!"
  }'

                        


                        

                            
פרמטרים נתמכים:

                            
    
                                
  • sessionId: מזהה החשבון השולח.
    • 
                                
  • to: מספר יעד (972...).
    • 
                                
  • message: תוכן ההודעה (טקסט חופשי).
    • 
                            

                        

                    


                    

                        

                             ניהול נעילת קבוצות
                        

                        

                            נעל או פתח קבוצות כך שרק מנהלים יוכלו לשלוח הודעות (או כל החברים).
                        


                        

                            
                            
                            

                                💡 טיפ: תוכל למצוא את מזהה הקבוצה ביומן הפעילות כאשר מתקבלת הודעה מהקבוצה
                            

                        


                        

                            
                            
                        


                        

                            

                                

                                     קבוצות
                                    בניהולך
                                

                                
                            

                            

                                

                                    לחץ על "רענן רשימה" כדי לראות קבוצות בניהולך

                            

                        


                        

                            

                                 API לנעילת קבוצות:
                            


                            

                                

                                    נעילת קבוצה (רק מנהלים):

                                
curl -X POST http://localhost:3000/api/group/lock \
  -H "Content-Type: application/json" \
  -d '{
    "sessionId": "default",
    "groupId": "120363123456789012@g.us"
  }'

                            


                            

                                

                                    פתיחת קבוצה (כל החברים):

                                
curl -X POST http://localhost:3000/api/group/unlock \
  -H "Content-Type: application/json" \
  -d '{
    "sessionId": "default",
    "groupId": "120363123456789012@g.us"
  }'
