🔌 תיעוד API

                        

                            ניתן לשלוח הודעות דרך ה-API החיצוני באמצעות בקשת HTTP POST.
                            הנה דוגמה לקוד עבור החשבון הנוכחי:
                        


                        

                            
                            
curl -X POST http://localhost:3000/api/send-message \
  -H "Content-Type: application/json" \
  -d '{
    "sessionId": "default",
    "to": "972501234567",
    "message": "Hello World!"
  }'

                        


                        

                            
פרמטרים:

                            
    
                                
  • sessionId - מזהה החשבון (חובה לשליחה מחשבון ספציפי)
    • 
                                
  • to - מספר טלפון בפורמט בינלאומי (ללא +)
    • 
                                
  • message - תוכן ההודעה
    •
