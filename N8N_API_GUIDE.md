# מדריך חיבור n8n לבוט הוואטסאפ 🤖🔗

הוספנו למערכת נקודת קצה (API Endpoint) המאפשרת ל-n8n (או לכל מערכת חיצונית) לשלוח הודעות וואטסאפ דרך הבוט שלך.

## כתובת ה-API
אם n8n רץ באותו דוקר (לפי ה-Bundle שלנו):
`http://whatsapp-bot:3000/api/send-message`

אם n8n רץ חיצונית:
`http://[YOUR_IP]:3000/api/send-message`

---

## איך לבנות את הבקשה ב-n8n (HTTP Request Node)

1. **Method:** `POST`
2. **URL:** `http://whatsapp-bot:3000/api/send-message`
3. **Authentication:** None (בשלב זה)
4. **Send Body:** `true`
5. **Body Content Type:** `JSON`
6. **Parameters:**
   - `to`: מספר הטלפון של הנמען (כולל קידומת מדינה, ללא +, למשל: `972501234567`).
   - `message`: הטקסט שברצונך לשלוח.

### דוגמה ל-Body (JSON):
```json
{
  "to": "972556609260",
  "message": "שלום! זו הודעה חוזרת מ-n8n אחרי ניתוח הנתונים."
}
```

---

## זרימת עבודה מומלצת (Workflow)
1. **Webhook Node:** הבוט שולח הודעה ל-n8n.
2. **AI / Logic Node:** n8n מעבד את המידע (למשל עם ChatGPT או לוגיקה פנימית).
3. **HTTP Request Node:** n8n שולח חזרה לבוט את התשובה הסופית דרך ה-API שכתוב למעלה.

---

## טיפים חשובים
- **פורמט המספר:** הבוט יודע להוסיף את ה-`@c.us` לבד, אז מספיק לשלוח את המספר בלבד.
- **סטטוס המערכת:** ההודעה תישלח רק אם הבוט במצב `READY` (מחובר לוואטסאפ).
- **לוגים:** תוכל לעקוב אחרי ההודעות שנשלחו חיצונית בתוך לשונית ה-Logs בממשק הניהול (יופיע עם אייקון 📤).
