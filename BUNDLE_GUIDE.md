# הבוט הכל-ב-אחד: WhatsApp + n8n 🚀

ברוך הבא! התיקייה הזו מכילה כעת "חבילה" שלמה של אוטומציה שרצה יחד.

## מה יש בחבילה?
1. **WhatsApp Bot UI** (פורט 3000) - ממשק הניהול, סריקת ה-QR, ניהול שלבים וסטטיסטיקה.
2. **n8n Automation** (פורט 5678) - המנוע הוויזואלי לאוטומציות מורכבות.

---

## איך מפעילים הכל בבת אחת? (Docker)

1. וודא שמותקן אצלך Docker ו-Docker Compose.
2. בטרמינל, הרץ את הפקודה:
   ```bash
   docker-compose up -d
   ```
   *הפרמטר `-d` דואג שהכל ירוץ ברקע בצורה שקטה.*

3. פתח את הדפדפן:
   - לבוט: `http://localhost:3000`
   - ל-n8n: `http://localhost:5678` (שם משתמש: `admin`, סיסמה: `admin_password_here`)

---

## חיבור בין הבוט ל-n8n
מכיוון שהם רצים יחד באותה חבילה ("Network"), הם יכולים לדבר אחד עם השני בקלות.
בכלל ה-Webhook בבוט, פשוט תשתמש בכתובת:
`http://n8n:5678/webhook/YOUR-HOOK-ID`

---

## פקודות ניהול
- **לראות לוגים של הכל:** `docker-compose logs -f`
- **לעצור את הכל:** `docker-compose down`
- **להפעיל מחדש אחרי שינויי קוד:** `docker-compose up -d --build`
